PassonOAS loop
    Events
        Time - Every 0.50 seconds of game time
    Conditions
    Actions
        For each (Integer PoaS_MUI3) from 1 to PoaS_MUI1, do (Actions)
            Loop - Actions
                -------- ai is for "anti-inconvenicence" (integer) --------
                Set VariableSet PoaS_ai = PoaS_MUI2[PoaS_MUI3]
                --------   --------
                Set VariableSet PoaS_mana = (Mana of PoaS_caster[PoaS_ai])
                If (All Conditions are True) then do (Then Actions) else do (Else Actions)
                    If - Conditions
                        (PoaS_caster[PoaS_ai] is in PoaS_activegroup.) Equal to True
                        (PoaS_caster[PoaS_ai] is alive) Equal to True
                        PoaS_mana Greater than or equal to 0.50
                    Then - Actions
                        Unit - Set mana of PoaS_caster[PoaS_ai] to ((Mana of PoaS_caster[PoaS_ai]) - 0.50)
                        -------- Area selection --------
                        Set VariableSet PoaS_tempp = (Position of PoaS_caster[PoaS_ai])
                        Set VariableSet PoaS_tempug = (Units within PoaS_aoe[PoaS_ai] of PoaS_tempp.)
                        Custom script:   call RemoveLocation(udg_PoaS_tempp)
                        -------- Caster healing --------
                        Unit - Set life of PoaS_caster[PoaS_ai] to ((Percentage life of PoaS_caster[PoaS_ai]) + 0.50)%
                        If (All Conditions are True) then do (Then Actions) else do (Else Actions)
                            If - Conditions
                                (PoaS_tempug is empty) Equal to False
                            Then - Actions
                                Unit - Set life of PoaS_caster[PoaS_ai] to ((Life of PoaS_caster[PoaS_ai]) + PoaS_dmg[PoaS_ai])
                            Else - Actions
                        -------- Area damage --------
                        Unit Group - Pick every unit in PoaS_tempug and do (Actions)
                            Loop - Actions
                                Set VariableSet PoaS_pickedu = (Picked unit)
                                If (All Conditions are True) then do (Then Actions) else do (Else Actions)
                                    If - Conditions
                                        (PoaS_pickedu is Magic Immune) Equal to False
                                        (PoaS_pickedu is dead) Equal to False
                                        (PoaS_pickedu belongs to an enemy of (Owner of PoaS_caster[PoaS_ai]).) Equal to True
                                    Then - Actions
                                        Unit - Cause PoaS_caster[PoaS_ai] to damage PoaS_pickedu, dealing PoaS_dmg[PoaS_ai] damage of attack type Spells and damage type Normal
                                        Special Effect - Create a special effect attached to the origin of PoaS_pickedu using Abilities\Spells\Human\Feedback\SpellBreakerAttack.mdl
                                        Special Effect - Destroy (Last created special effect)
                                    Else - Actions
                        Custom script:   call DestroyGroup(udg_PoaS_tempug)
                        -------- SpeedBoost --------
                        Set VariableSet PoaS_dur[PoaS_ai] = (PoaS_dur[PoaS_ai] - 0.50)
                        If (All Conditions are True) then do (Then Actions) else do (Else Actions)
                            If - Conditions
                                PoaS_dur[PoaS_ai] Less than or equal to 0.00
                            Then - Actions
                                Unit - Remove PassionOfASaint BuffAura (Endurance Dummy) from PoaS_caster[PoaS_ai]
                            Else - Actions
                    Else - Actions
                        -------- No longer active --------
                        Unit Group - Remove PoaS_caster[PoaS_ai] from PoaS_activegroup.
                        Special Effect - Destroy PoaS_attach[PoaS_ai]
                        Unit - For PoaS_caster[Dash_ai], Ability Swordsaint: Passion of a Saint (Channel ON), Hide ability: False
                        Unit - For PoaS_caster[Dash_ai], Ability Swordsaint: Passion of a Saint (Channel OFF), Hide ability: True
                        Unit - Remove PassionOfASaint BuffAura (Endurance Dummy) from PoaS_caster[PoaS_ai]
                        -------- recycle indices --------
                        Set VariableSet PoaS_MUI2[PoaS_MUI3] = PoaS_MUI2[PoaS_MUI1]
                        Set VariableSet PoaS_MUI2[PoaS_MUI1] = PoaS_ai
                        Set VariableSet PoaS_MUI1 = (PoaS_MUI1 - 1)
                        Set VariableSet PoaS_MUI3 = (PoaS_MUI3 - 1)
                        If (All Conditions are True) then do (Then Actions) else do (Else Actions)
                            If - Conditions
                                PoaS_MUI1 Equal to 0
                            Then - Actions
                                Trigger - Turn off (This trigger)
                            Else - Actions
