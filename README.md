Scripts and design docs for an ensemble of Heroes I felt like making for Blizzard's Warcraft III RTS game.
Everything here is in lua - not because I'm amazingly profficient with that, but because it was a fun way to learn a new language.

Feel free to steal/copy/take-as-inspiration/loan/pilfer or borrow from it as you please.
All of the abilities here rely on Bribe of Hiveworkshop.com's [Total Initialization](https://www.hiveworkshop.com/threads/total-initialization.317099/) system to create the triggers.

See [this repository](https://github.com/Woodenplank/Lua-LibStubs/tree/main) for all necessary "mini-libraries" I made for these Hero setups.

Beware of outstanding issues! Usually noted down with --inline code comments.
Use at your own risk.
